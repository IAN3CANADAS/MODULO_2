FING - UNCUYO

SUCESIÓN FIBONACCI


Es una serie infinita de números naturales donde cada número se obtiene sumando los dos anteriores.
Comienza con los números 0 y 1, a partir de estos cada uno de los siguientes términos es la suma de los dos anteriores.(0,1,1,2,3,5,8,13…)


Es muy famosa porque aparece constantemente en la naturaleza (piñas,girasoles,ramas de árboles,caracoles,etc.) y tambien se aplica en el arte y la arquitectura para lograr proporciones armoniosas.

Secuencia Fibonacci y La Proporción Áurea

Si tomamos dos números consecutivos de la serie de Fibonacci y dividimos el mayor por el menor, el resultado se aproxima cada vez más a ϕϕ=1,618033 conocida como “Proporción Áurea”

A continuación puede verse el código para implementar la serie:

# 1. Configuración de variables iniciales
n <- 100
F <- vector(mode = "numeric", length = n)
A <- 0
B <- 1

# 2. Asignación de los dos primeros valores
F[1] <- A
F[2] <- B

# 3. Bucle para calcular la secuencia de Fibonacci
for (i in 3:n) {
  F[i] <- F[i-1] + F[i-2]
}

# 4. Mostrar las primeras 6 posiciones del vector
head(F)
[1] 0 1 1 2 3 5
# Inicialización
F <- c(0, 1)
i <- 2

# Bucle que se detiene al superar 1.000.000
while (F[i] <= 1000000) {
  proximo <- F[i] + F[i-1]
  F <- c(F, proximo)
  i <- i + 1
}

# Resultados
cat("El primer número mayor a 1M es:", F[i], "\n")
El primer número mayor a 1M es: 1346269 
cat("Se necesitaron", i, "iteraciones (elementos del vector).")
Se necesitaron 32 iteraciones (elementos del vector).
